## 📚 제가 참여하는 과정은?

제가 참여하는 **코디세이 AI 네이티브 과정**은 개발 지식이 없어도 AI를 활용해 문제를 해결하는 방법을 배우는 5개월간의 몰입형 교육입니다.

- **기간**: 2026년 6월 1일(월) ~ 10월 30일(금)
- **학습 방식**:
    - **강의 없는 동료학습**: 정해진 강의나 교재 없이, 동료들과 함께 기획부터 서비스 배포까지 모든 과정을 경험합니다.
    - **실무 프로젝트 중심**: 생성형 AI, 노코드, Python, API 연동 등을 활용해 실제 서비스를 만들어보는 프로젝트를 진행합니다.
    - **자기주도적 시간 관리**: 월 80시간 이상 참여를 목표로, 스스로 학습 계획을 세우고 실행합니다.
- **학습 공간**: 24시간 언제나 열려있는 캠퍼스(서울 강남)에서 학습에 몰입할 수 있습니다.
